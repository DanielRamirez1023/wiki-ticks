VISUAL STUDIO CODE

Herramienta de Desarrollo

¿Qué es?

Visual Studio Code (VS Code) es un editor de código fuente desarrollado por Microsoft. Es software libre y multiplataforma, está disponible para Windows, GNU/Linux y macOS.

Beneficios

  • Es multiplataforma.
  • Uso de Extensiones.
  • Uso de depuración para detectar errores.
  • Uso del control de versiones.
  • Te permite sistematizar todos tus diseños y alojarlos en la nube

¿Qué lenguajes soporta VS Code?

  • Principalmente soporta JavaScript/TypeScript, Node.js

Características de VS Code

  • Es personalizable con los cientos de plugins o extensiones disponibles para escribir código en diferentes lenguajes.
  • Disponible para todos los Sistemas Operativos
  • VS code tiene una terminal integrada que puede utilizar cualquier Shell instalado en el equipo, como PowerShell, Bash o cualquier otro.
