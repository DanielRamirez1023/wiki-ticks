DRAW IO

Software de Diagramas UML, ER, diagramas red en línea

¿Qué es?

Draw.io es una herramienta que permite realizar cualquier tipo de diagrama de flujo, diagramas de procesos, organigramas, así como diagramas de red, UML, mapas conceptuales y otros elementos necesarios para realizar un diseño.

Beneficios

  • Permite diseñar Diagramas online
  • Es totalmente gratis
  • Facilita el uso de herramientas básicas y sencillas como el sistema drag and drop
  • Está disponible para la descarga en su versión offline, que permite el trabajo sin conexión

Conocimientos Necesarios

  • Diagramas UML, de flujo, de clases, entre otros.
