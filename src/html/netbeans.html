APACHE NETBEANS

Herramienta de Desarrollo

¿Qué es?

NetBeans es un entorno de desarrollo integrado libre, hecho principalmente para el lenguaje de programación Java.

Beneficios

  • Es multiplataforma.
  • Código abierto.
  • Manejo automático de la memoria.
  • Multilenguaje

¿Qué lenguajes soporta NetBeans?

  • Principalmente Java pero NetBeans puede utilizar otros lenguajes como PHP, C o Ruby, entre otros.

Características de NetBeans

  • Proporciona una base modular y extensible para el desarrollo de aplicaciones.
  • Disponible para todos los Sistemas Operativos
  • Esta plataforma incluye servicios para el control del interfaz de usuario, la configuración, el almacenamiento, las ventanas, etc
