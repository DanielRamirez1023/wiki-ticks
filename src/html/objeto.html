GIT

Sistema de control de versiones Git

Que Es?

Git, es un software de control de versiones,se define como control de versiones a la gestión de los diversos cambios que se realizan sobre los elementos de algún producto o una configuración del mismo es decir a la gestión de los diversos cambios que se realizan sobre los elementos de algún producto o una configuración,

Beneficios

  • Permite controlar versiones locales y globales
  • Mejora y facilita el trabajo en equipo
  • Puedes Hacer colaboraciones a proyectos OpenSource
  • Es multiplataforma por lo que se puede usar para crear repositorios locales en todos los sistemas operativos más comunes, como Windows, Linux o Mac.
  • Lo utilizan empresas tecnológicas de referencia como Google, Facebook o Nefflix para controlar las versiones de código fuente sus proyectos.

Conocimientos Necesarios

  • Navegación básica en terminal o linea de comandos
