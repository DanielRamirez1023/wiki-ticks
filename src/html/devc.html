Herramienta de Desarrollo

¿Qué es?

Eclipse es un entorno de desarrollo software multi-lenguaje construido alrededor de un workspace al que pueden incluirse un gran número de plug-ins que proporcionan funcionalidades concretas relacionadas con lenguajes específicos o con la interacción con otras herramientas implicadas en el desarrollo de una aplicación.

Beneficios

  • Es multiplataforma.
  • Herramienta de código abierto.
  • Proporciona una plataforma RCP (Rich Client Plataform) para el desarrollo de aplicaciones de propósito general.
  • Permite el desarrollo de aplicaciones para el servidor y servicios web.

¿Qué lenguajes soporta Eclipse?

  • Principalmente Java pero puede utilizar otros lenguajes como ANSI C, C++, JSP, entre otros.

Características de Eclipse

  • Tiene pruebas unitarias con JUnit, control de versiones con CVS, integración con Ant, asistentes (wizards) para creación de proyectos, clases, tests, etc., y refactorización.
